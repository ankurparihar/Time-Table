No cell selected

Instructions and Features

  • Select single cells by clicking
  • Select multiple cells by clicking and moving
  • If selected cells are of same type [data or header], then options are displayed
  • Copy [button or ctrl+c] and Paste [button or ctrl+v] single cell
  • Copy [single cell], Paste [multiple cells], Clear [multiple cells]
  • Batch select or clear modes
  • Merge cells row-wise or column-wise, called spanning
  • Right now table is configured using timetable_data in script.js
  • Right now screenshot is the only way to save this
